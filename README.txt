FOLDER STRUCTURE:

audio/
  - rustle.mp3
  - content-image-sound-1.mp3 through content-image-sound-6.mp3

images/
  - nav/ (navigation sidebar images)
    - nav1.jpg through nav6.jpg
  - content/ (clickable content page images)
    - content1.jpg through content6.jpg

root pages/
  - index.html (page 1)
  - page2.html through page6.html
    - fully separate HTML pages with shared nav

AUDIO FILES NEEDED:

1. rustle.mp3 — Optional paper rustle/crinkle sound file. This file is not currently linked to page navigation, but it can be used if you want a click sound for nav items.

2. content-image-sound-1.mp3 — Audio for content image 1 (e.g., "hello")
3. content-image-sound-2.mp3 — Audio for content image 2
4. content-image-sound-3.mp3 — Audio for content image 3
5. content-image-sound-4.mp3 — Audio for content image 4
6. content-image-sound-5.mp3 — Audio for content image 5
7. content-image-sound-6.mp3 — Audio for content image 6

Each content image has its own audio file with the naming pattern above.

Supported formats: MP3, WAV, OGG.

Tips:
- Keep audio files short (1-2 seconds) for quick playback
- Files can be different lengths
- Find free sound effects at freesound.org or create your own
- Rename files exactly as shown above (important for the code to work)
